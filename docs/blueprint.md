# **App Name**: FAITH Connect

## Core Features:

- Member Directory: A searchable directory of church members with fields for first name, last name, email address, phone number, birthday, status (Active, Prospect, Archived), notes, and a profile photo.
- Member Management: Enable adding, editing, and deleting member profiles. Enforce required fields for new member profiles.
- Event Calendar: A monthly calendar view to add and display church events on specific days.
- Contribution Tracking: Record member contributions, categorized as Tithes, Offering, Donation, and Other. Display charts showing recent contribution amounts by category.

## Style Guidelines:

- Background color: Black (#000000) for a sleek and modern base.
- Primary color: White (#FFFFFF) for high contrast and readability of text and main elements.
- Accent color: Slate gray (#708090) for subtle highlights and interactive elements.
- Body and headline font: 'Inter', a sans-serif font, provides a clean, modern, neutral look suitable for both headlines and body text.
- Minimalist design with plenty of whitespace to provide clarity and focus.
- Simple, line-based icons for navigation and key actions. The icons will be a lighter shade of gray to contrast with the background.
- Subtle transitions and animations to enhance user experience without being distracting.
